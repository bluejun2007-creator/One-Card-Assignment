# One-Card-Assignment

파이썬으로 구현한 AI와 1:1로 즐길 수 있는 원카드 게임입니다.

---

## 주요 기능
* **객체지향 설계**: `Card`, `Deck`, `Player`, `OneCardGame` 클래스로 나누어 가독성과 확장성을 높였습니다.
* **예외 처리**: 유저가 잘못된 숫자나 문자를 입력했을 때 게임이 비정상 종료되지 않도록 예외 처리를 반영했습니다.
* **자동 재셔플**: 덱의 카드가 모두 소진되면, 기존에 바닥에 쌓였던 카드더미를 자동으로 다시 섞어 덱을 재구성합니다.

---

## 게임 규칙

> **최종 목표**: 자신의 손에 든 모든 카드를 상대방보다 먼저 바닥에 내려놓는 플레이어가 승리합니다.

* **1. 게임 준비**
    * 게임 시작 시, 유저와 컴퓨터는 각각 **7장의 카드**를 무작위로 분배받습니다.
    * 남은 카드 더미(Deck)에서 맨 위의 카드 1장을 뒤집어 **바닥(Discard Pile)**에 오픈하며 게임이 시작됩니다.

* **2. 카드 내기 규칙**
    * 플레이어는 바닥에 놓인 최상단 카드와 **문양(Shape)** 또는 **숫자/알파벳(Rank)**이 일치하는 카드만 제출할 수 있습니다.
    * 예시: 바닥 카드가 `♠7`인 경우, `♠ 문양의 카드` 전체 또는 다른 문양의 `7 카드`만 낼 수 있습니다.

* **3. 턴 진행 및 드로우**
    * 자신의 차례에 낼 수 있는 카드가 없거나, 전략적으로 카드를 내지 않으려면 **숫자 `0`을 입력하여 카드 1장을 뽑고(Draw)** 턴을 넘겨야 합니다.
    * 카드를 제출하는 데 성공하면 즉시 상대방에게 턴이 넘어갑니다.
